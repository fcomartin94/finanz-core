Finanz API

API REST para gestión de presupuesto personal. Haz clic en los enlaces para probar:

GET /api/saldo — Saldo total
GET /api/saldo/mes-actual — Saldo del mes
GET /api/resumen/mes-actual — Resumen mensual
GET /api/transacciones — Listar transacciones

Para registrar transacciones usa POST /api/transacciones/simple con curl o Postman.
